# Lead Executive Dashboard

A standalone HTML dashboard for tracking lead generation and conversion metrics.

## Features

- **Real-time Metrics Display**: Total leads, conversion rate, active campaigns, and revenue
- **Interactive Charts**: Visual representation of leads by source and status distribution
- **Recent Activity Feed**: Track latest lead activities and conversions
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Dependencies**: Single HTML file with embedded CSS and JavaScript

## Usage

Simply open `index.html` in any modern web browser. No installation or server required.

### Quick Start

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Click the "Refresh Dashboard" button to see updated metrics

## Features Included

- Animated metric counters
- Interactive bar charts
- Activity timeline with status indicators
- Live data refresh functionality
- Mobile-responsive layout
